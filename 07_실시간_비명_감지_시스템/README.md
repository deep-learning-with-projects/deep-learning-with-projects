# Scream Detection using CNN
CNN을 활용하여 비명인지 비명이 아닌지 Binary Classification 하는 프로젝트입니다.

* [Dependency Install](#Dependency-Install)
* [데이터 다운로드 가이드](#데이터-다운로드-가이드)

## Dependency Install
```shell
pip3 install -r requirements.txt
```
